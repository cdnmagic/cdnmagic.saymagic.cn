# 需要注意

1. 进来页面之前做一个是否登录的验证, 没有则转到登录页面
2. 当设备处于"开始推送"状态, 则无法对"推送规则"进行编辑
3. 推送规则 的状态
    未推送 --> 开始推送
    已完成 --> 再次推送
    未开始
4. 各种 API 均有报错信息.
